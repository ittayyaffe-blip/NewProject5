# דאשבורד אינטרקטיבי — פרחי איריס

אפליקציית ווב אינטראקטיבית לחקירת מאפייני פרחי האיריס (Iris), בנויה עם **Dash** ו-**Plotly**.

## תיאור

הפרויקט מציג גרף פיזור (scatter plot) דינמי שמאפשר לבחור תכונות לציר X ולציר Y מתוך מערך הנתונים הקלאסי של Iris. הצבעים בגרף מייצגים את שלושת סוגי הפרח.

## תכונות

- בחירת תכונה לציר X מתוך רשימה נפתחת
- בחירת תכונה לציר Y מתוך רשימה נפתחת
- עדכון מיידי של הגרף בעת שינוי הבחירה
- צביעה לפי סוג פרח (`species`)
- ממשק בעברית

## דרישות מערכת

- Python 3.8 ומעלה
- החבילות: `dash`, `plotly`, `pandas`

## התקנה

1. שכפל או הורד את הפרויקט:

```bash
git clone <כתובת-המאגר>
cd NewProject2
```

2. (מומלץ) צור סביבה וירטואלית:

```bash
python -m venv venv
venv\Scripts\activate
```

3. התקן את התלויות:

```bash

```

```bash

```

פתח בדפדפן את הכתובת שמוצגת בטרמינל (בדרך כלל `http://127.0.0.1:8050`).

## איך זה עובד

1. הנתונים נטענים מ-`plotly.express.data.iris()`.
2. שני רכיבי `dcc.Dropdown` מאפשרים לבחור עמודות מספריות לצירים.
3. callback של Dash מעדכן את `dcc.Graph` בכל שינוי בבחירה.
4. `plotly.express.scatter` יוצר את הגרף עם צביעה לפי `species`.
